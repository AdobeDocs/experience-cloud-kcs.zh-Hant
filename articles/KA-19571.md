# 如何檢查由 Service Pack 安裝更新的節點

## 說明


安裝 AEM 6.5.12.0 等 Service Pack 時，您可能想知道哪些節點已更新。本文介紹如何使用偵錯記錄檢查這些節點。

<b>環境</b>
AEM 6.5


## 解析度


安裝Service Pack之前，請為org.apache.jackrabbit.vault.packaging.impl.ActivityLog設定除錯記錄。 此偵錯紀錄會輸出更新的節點路徑。

- 1前往Web主控台Sling記錄支援(http://localhost:4502/system/console/slinglog)
- 2選擇「添加新記錄器」按鈕
- 3填入下列值，然後選取「儲存」按鈕

配置示例：      loglevel = DEBUG logfile = logs/error.log logger = org.apache.jackrabbit.vault.packaging.impl.ActivityLog

- 4安裝Service Pack
- 5檢查除錯記錄
