# 如何檢查由Service Pack安裝更新的節點

## 說明

安裝AEM 6.5.12.0等Service Pack時，您可能想知道已更新的節點。 本文說明如何使用偵錯記錄來檢查這些節點。

<b>環境</b>

AEM 6.5

## 解析度

安裝Service Pack之前，請先為 `org.apache.jackrabbit.vault.packaging.impl.ActivityLog`. 除錯記錄會輸出更新的節點路徑。

1. 前往Web主控台Sling記錄支援(http://localhost:4502/system/console/slinglog)
1. 選擇「添加新記錄器」按鈕
1. 填入下列值，然後選取「儲存」按鈕

   設定範例:

   ```
   
   
  loglevel = DEBUG logfile = logs/activity.log logger = org.apache.jackrabbit.vault.packaging.impl.ActivityLog

```
1. Install the service pack
1. Check the debug log
