# 今天日期的Data Warehouse請求是否可傳回目前資料？ | [!DNL Analytics]

## 說明


<b>環境</b>
Data Warehouse

<b>問題</b>
今天日期的Data Warehouse請求是否可傳回目前資料？


## 解析度


<b>回答</b>
可以，您可以排程每小時Data Warehouse請求，以傳送相同日期（當天）的每小時檔案，以傳回目前的資料。
