# 如何在設定檔指令碼中建立Cookie值？

## 說明


<b>環境</b>

Target



<b>問題/症狀</b>

是否可使用設定檔指令碼建立Cookie值，其中在傳回前有Cookie存在 *true/false* 是否已檢查？

此 `user.cookie` 嘗試此方法時，語法無法運作。


## 解析度


伺服器端存在設定檔指令碼，用戶端則存在Cookie。 若要使用設定檔指令碼建立Cookie值，請使用 `mboxParameters`.
