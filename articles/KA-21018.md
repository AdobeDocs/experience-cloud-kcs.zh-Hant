# Adobe Experience Manager Forms:如何解決PDF產生器錯誤，JPEG為2000

## 說明 {#description}


在具有Acrobat 2020 Classic的Windows伺服器上，PDF產生器無法處理JPEG2000格式資產(JPF、JPX、JP2、J2K、J2C、JPC)，並出現下列錯誤。


```
Error Code:1011 
Error Message: ALC-PDG-001-011-The BMC reported failure in converting the input document to PDF.
```


本文說明如何解決此問題。

<b>環境</b>

AEM 6.5 JEE Acrobat上的Forms 2020傳統版Microsoft Windows Server 2019


## 解析度 {#resolution}


設定Acrobat的註冊表項目。

1. 在運行AEM Forms的Windows Server上開啟註冊表編輯器

2. 選取HKEY_CURRENT_USER\SOFTWARE\Adobe\Adobe Acrobat\DC\ImageConversionPlugin下方的路徑

3. 建立新索引鍵（如果buseSandbox存在，則僅變更資料）

名稱：buseSandbox類型：REG_DWORD（32位）值資料：0

從PDF產生器重新執行「建立PDF」，並檢查PDF產生是否成功。
