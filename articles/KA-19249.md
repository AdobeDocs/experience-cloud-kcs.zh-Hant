# 如何以JSON格式匯出資產的本機參考資料

## 說明


您可以在上看到資產的本機參考 [!UICONTROL 參考] 標籤 [!UICONTROL 屬性]. (輸入 [!DNL Assets] 主控台，選取資產，選取 [!UICONTROL 屬性] ，然後選取 [!UICONTROL 參考] 標籤。) 使用AEM以外的資訊時，可以匯出為JSON格式。

<b>環境</b>

AEMas a Cloud Service, AEM 6.5


## 解析度


存取下列URL&#39;1&#39;會傳回JSON格式的本機參考。 `path` 回應中的值是參考目標資產的內容路徑。

&#39;1&#39; http://localhost:4502/bin/wcm/references.json?path=path以鎖定資產

<b>使用範例：</b>
http://localhost:4502/bin/wcm/references.json?path=/content/dam/wknd/en/activities/hiking/equipment_6.jpg
