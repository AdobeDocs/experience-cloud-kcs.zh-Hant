# 擴充/查詢活動：「所有行」選項提供錯誤

## 說明


<b>環境</b>
Adobe Campaign Classic

<b>問題</b>
使用 <b>所有列</b> 包含擴充或查詢活動的選項會導致錯誤：


```
BAS-010000 Unable to complete the operation in current status.
```



## 解析度


<b>解決方案</b>
此 <b>所有列</b> 按一下 <b>完成</b> (而非 <b>下一個</b>)，然後執行工作流程。

UI似乎有次要問題，而 <b>下一個</b> 按鈕在變更為 <b>所有列</b> 選項。
