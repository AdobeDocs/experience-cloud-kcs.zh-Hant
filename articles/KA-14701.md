# Adobe Campaign：錯誤：無法剖析運算式&#39;[@registrationToken]『

## 說明

據報導，嘗試使用行動應用程式頻道時，擲回以下錯誤：

```
error XTK-170036 Unable to parse expression '@registrationToken'
```

## 解決方法


解決方法是安裝 [!UICONTROL 行動應用程式頻道] 在執行例項上。
