# A4T活動中報表對象的成功體驗不同

<b>問題/症狀</b>

A4T中報表對象的成功體驗為何不同 [!DNL Target] 活動報表？




## 解析度


- 全域獲勝者不同的原因，是根據擷取的預設/主要目標量度值來決定 [!DNL Target]，而報告獲勝者則完全根據 [!DNL Analytics].
- 理想情況下，兩者應相同，但鑑於資料可能有所差異，兩者可能有所差異。
