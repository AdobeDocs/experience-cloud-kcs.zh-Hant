# A4T活動中報表對象的成功體驗不同

## 說明


<b>環境</b>

Target



<b>問題/症狀</b>

為什麼A4T中報表對象的成功體驗不同 [!DNL Target] 活動報表？




## 解決方法


- 全域獲勝者與眾不同的原因是，其決定取決於中擷取的預設/主要目標量度值。 [!DNL Target]，而報表獲勝者完全根據從取得的資料決定 [!DNL Analytics].
- 理想情況下，兩者應該相同，但由於資料可能有所差異，因此兩者可能有所差異。
