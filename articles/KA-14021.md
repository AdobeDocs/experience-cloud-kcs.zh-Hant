# A4T活動中報表對象的成功體驗不同

## 說明


<b>環境</b>

Target



<b>問題/症狀</b>

A4T Target活動報表中的報表對象為何有不同的成功體驗？




## 解析度


- 全域獲勝者不同的原因在於，其決定是根據Target中擷取的預設/主要目標量度值，而報表獲勝者則完全根據從Analytics取得的資料。
- 理想情況下，兩者應相同，但鑑於資料可能有所差異，兩者可能有所差異。
