# OOTB將DAM資產匯出至 [!DNL Target]

## 說明

是否有任何OOTB方式可將DAM資產從AEM匯出至 [!DNL Target]?<br><br><br>

## 解析度


沒有OOTB選項可將資產匯出至 [!DNL Target]. 只能將XF導出到 [!DNL Target].



由於AEM應是您的集中式DAM，因此最好在AEM中托管資產，並使用資產的連結來取代中的資產 [!DNL Target] 以避免在多個來源中重複。
