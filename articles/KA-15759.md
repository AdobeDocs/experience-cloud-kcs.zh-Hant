# 如何查找 [!DNL Launch] 暗旗？

## 說明

如何知道我目前擁有的AdobeAnalysis Workspace功能？ 

## 解析度

[!DNL Launch] 黑暗旗標會設定IMS組織層級。

登入 [!DNL Analytics]，然後前往 [!UICONTROL 工作區].

* [!UICONTROL 工作區] > [!UICONTROL 說明] > [!UICONTROL 關於工作區]
* 看陰性特徵標幟
