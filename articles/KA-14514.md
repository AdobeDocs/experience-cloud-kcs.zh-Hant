# 活動依據 [!UICONTROL 時間範圍] 目標定位無法運作

## 說明


<b>環境</b>
Adobe Target

<b>問題</b>
根據時間範圍鎖定目標的活動無法正常運作。

<b>複製步驟：</b>

1. 建立只在星期三和星期日執行的活動。
2. 將時間範圍納入：

   1. [!UICONTROL 週與日劃分]
   2. 星期三 —  [!UICONTROL 全天]
   3. 星期日 —  [!UICONTROL 全天]




<b>預期結果：</b>
活動會如預期般執行。 此活動每週三和週日的1時整。

<b>實際結果：</b>
由於此設定，活動只會在星期三和星期日執行。

活動執行時，會發生約15-20分鐘的延遲。


## 解析度


<b>解決方案</b>
需要在 <b>[!UICONTROL 時間範圍]</b> 觀眾是GMT +5次。

範例：如果您需要在星期五00:00到星期五23:59之間傳送活動，則可將時間設定為星期四23:30到星期五：23點30分。
