# 將目標A/B測試大小縮小

## 說明


如果您傳送「主旨型A/B」測試，且發現標示為警告的Target大小太小（沒有檔案提及最小目標大小）。 未選取成功者時，會手動選取成功變體。




| 測試母體提取：2種變體中每一種的1622條報文(總人數32451) |
| --- |
| 測試區段大小對A/B測試而言太小：「1622」建議最少為「3245」 |



## 解析度


這是進行AB測試的重要性。 如果總人口少於5萬，則學習人口至少必須佔總人口的10%。
