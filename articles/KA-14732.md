# [!UICONTROL Recommendations] 條件顯示錯誤的結果

## 說明


<b>環境</b>

Target



<b>問題/症狀</b>

[!UICONTROL Recommendations] 的CSV檔案。


## 解析度


<b>原因</b>

由於的包含規則，可能會篩選結果 `primaryCategory`.



<b>Recommendations</b>

1. 檢閱活動定義和條件定義。
2. 從CSV下載項目中檢查個別環境的結果。
3. 檢查上述任何結果中是否包含包含規則。 否則，會向訪客顯示備份建議。
