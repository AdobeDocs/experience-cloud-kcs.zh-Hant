# 非預期的AAM UI存取權

## 說明 {#description}


<b>環境</b>

- Audience Manager
- 客戶屬性
- 對象庫


<b>問題/症狀</b>



使用者可以存取AAM UI，但是他們沒有特定的AAM許可權


## 解決方法 {#resolution}


這是因為客戶屬性和對象庫許可權：

![](assets/0f984131-f8d2-ed11-a7c7-6045bd006b25.png)



後端採用Audience Manager技術，讓客戶屬性和對象庫正常運作，因此除了這些許可權集外，也會授予有限的AAM許可權集。 因此，如果您不希望使用者存取AAM UI，您也將需要移除其對象庫和客戶屬性許可權。
