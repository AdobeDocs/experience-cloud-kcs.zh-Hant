# [!UICONTROL 檔案收集器] ACS中的活動替代

## 說明

因為我們沒有 [!UICONTROL 檔案收集器] 活動，您可以使用下列活動來構建它，並且您將獲得或多或少相同的功能

## 解析度


![](assets/8e624e34-2cce-ec11-a7b5-0022480a8e40.png)



此 [!UICONTROL 檔案收集器] 將允許根據外部伺服器的檔案名稱模式下載多個檔案。 每個檔案都會傳遞至 [!UICONTROL 載入檔案] 活動，將資料提供至更新。
完成後，會繼續使用下列檔案等。
檔案傳輸也具有 [!UICONTROL 歷史化設定]如您所知 [!UICONTROL 檔案收集器].
