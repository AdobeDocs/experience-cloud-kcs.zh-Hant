# Analytics Workspace篩選器和/或區段的逗留時間

## 說明

如何以處理篩選器/區段 [!UICONTROL 逗留時間] 量度？

## 解析度


執行報表時：

- 系統會針對所選日期範圍收集資料
- 會計算逗留時間值
- 群體會篩選結果


範例影響：由於逗留時間值是在執行篩選器/區段之前分配，因此可能會有沒有造訪的頁面（維度）逗留時間。
