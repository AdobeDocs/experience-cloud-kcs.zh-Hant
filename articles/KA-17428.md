# 管理status.adobe.com的訂閱

## 說明 {#description}

<b>環境</b>
- Experience Cloud



<b>問題/症狀</b>
管理status.adobe.com通知訂閱或訂閱的步驟。


## 解析度 {#resolution}


1. 前往status.adobe.com
2. 使用Adobe ID登入
3. 按一下「管理訂閱」
4. 按一下「建立訂閱」
5. 在「Experience Cloud」下 — 選擇要訂閱的解決方案和子元件 — 繼續
6. 選擇地區和事件類型 — 繼續
7. 確認訂閱，然後按一下完成
