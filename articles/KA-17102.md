# 分類API匯入錯誤關係0目前未設定為匯入資料

## 說明


分類(SAINT)API中API匯入錯誤「關係0目前未設定為匯入資料」的原因為何？


## 解析度


將變數&quot;product&quot;與大寫的&quot;Product&quot;搭配使用時，會出現「Relation 0 is not set to import data」錯誤。 分類API只會接受小寫為「product」且為「p」的變數。
