# Analysis Workspace中不提供「比較時段」

## 說明


為何Analysis Workspace中可能無法使用「比較時段」功能？


## 解析度


Analysis Workspace中的「比較時段」功能僅適用於具有進階計算量度之SKU的公司。 換言之，依預設，這些SKU適用於Select、Prime和Ultimate的公司。 「Foundation」SKU上的公司無法使用。

此外，當時間相關元件（日、月等） 會新增為維度。
