# 提升度值從 [!DNL Target] UI

## 說明


<b>問題：</b> 我可以在 [!DNL Target] UI，但檢查相同日期範圍時，現在無法使用


## 解析度




<b>事實：</b> UI不會顯示體驗的提升度（可信度為0）



<b>說明： </b>當UI顯示提升度時，可信度的值為0。 不過在當天較晚的時候，當有更多訪客時，信賴度會降為0
