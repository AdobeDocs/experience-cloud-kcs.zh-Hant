# 提升度值從Target UI中消失


提升度會顯示在Target UI中（如果可信度） `>`  0. 如果可信度，Target UI中不會顯示期限 `<`  0.

## 說明 {#description}


### <b>環境</b>

- Target


### <b>問題/症狀</b>

提升度值可在Target UI中使用，但現在檢查相同日期範圍時也可使用。


## 解決方法 {#resolution}




<b>事實：</b> UI不會顯示可信度體驗的提升度 `<`  0



<b>說明： </b>當UI顯示提升度時，可信度的值為 `>`  0. 然而，當天稍後隨著更多訪客，信心降低為 `<`  0
