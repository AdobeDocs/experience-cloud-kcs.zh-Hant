# 建立特徵以擷取變數c_evar1中的所有值

## 說明 {#description}

<b>環境</b>
Audience Manager


<b>問題/症狀</b>
如何建立特徵來擷取變數中所有傳入的值 `c_evar1` 在Adobe Audience Manager(AAM)中？


## 解析度 {#resolution}


這是在 *規則運算式。*

條件，例如 `(c_evar1 matchesregex "*.*$")` 會建立特徵來擷取所有傳入AAM的值。 這就像 `c_evar1` 存在。
