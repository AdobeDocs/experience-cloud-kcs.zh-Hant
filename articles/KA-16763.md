# 建立特徵以擷取變數c_evar1中的所有值

## 說明

<b>環境</b>
Audience Manager
<b>問題/症狀</b>
如何在Adobe Audience Manager(AAM)中建立特徵來擷取所有傳入值？


## 解析度


這是在 *規則運算式*.

條件，例如 `(c_evar1 matchesregex "*.*$")` 會建立特徵來擷取所有傳入AAM的值。 這將起作用 *如果c_evar1存在*.
