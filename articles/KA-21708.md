# 更新「僅限結構」不會更新語言副本的內容

## 說明 {#description}

<b>環境</b>
Adobe Experience Manager(AEM)
<b>問題/症狀</b>
使用 <b>僅限結構</b> 選項時，不會更新頁面的內容。

<b>要再現的步驟</b>

1. 使用 <b>僅限結構</b> 選項。
2. 通過添加文本來編輯英文/主版頁。
3. 請使用 <b>僅限結構</b> 選項。
4. 請注意，步驟2中所做的變更不會更新。



## 解決方法 {#resolution}


如果需要內容變更，使用者必須手動促銷啟動。
<b>原因</b>
這是預期的行為。 <b>僅限結構</b> 更新不會促銷啟動，因此內容不會更新。
