# 即使訪客正確符合活動資格（非A4T），活動報表也未填入

## 說明


<b>環境</b>

Target



<b>問題/症狀</b>

即使訪客符合活動資格，系統也不會在活動報表中擷取訪客。


## 解析度


如果您檢閱的報表環境不正確，就可能會發生此情況。



在這種情況下，請檢閱URL的環境，並選取正確的環境以擷取報表。
