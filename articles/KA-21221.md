# 無法儲存相似模型


瞭解如何解決Adobe Audience Manager (AAM)中的相似模型無法儲存的問題。

## 說明 {#description}


### 環境

Audience Manager

### <b>問題/症狀</b>

在AAM中使用相似模型時，它拒絕儲存。


## 解決方法 {#resolution}


如果有問題的模型使用來自 Marketplace 的資料，請確保您在模型中使用的資料已啟用模組化功能。如果沒有，就不會儲存相似模型。如果不是這個問題，請開啟支援案例。
