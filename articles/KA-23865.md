# CTT （內容轉移工具）在雲端例項之間是否有效


本文探討內容轉移工具(CTT)是否可以在AEMas a Cloud Service的雲端例項之間移轉現有內容。

## 說明 {#description}


### 環境

AEM as a Cloud Service

### 問題/症狀

內容轉移工具(CTT)是否適用於雲端例項？


## 解決方法 {#resolution}


若要讓CTT將現有內容移轉至目標AEM Cloud Service例項，必須有非雲端AEM來源例項（內部部署或AMS）。 因此，不行，CTT無法在雲端例項之間運作。
