## 解決方法 {#resolution}


此故障是客戶代碼故障，需要使用 npm 更新故障模組中的節點版本來解決。要進行此變更，您的客戶需要執行以下操作：

1. 找到反應器 pom.xml 中使用的節點版本。下列範例 = 16.14.0    `pom.xml`:


   ```
   configuration
           nodeVersionv16.14.0/nodeVersion. 
           npmVersion8.3.1/npmVersion
       configuration
   ```

2. 將此節點版本作為 <b>devDependencies</b> 新增到組建失敗模組的 `package.json` 檔案中。在上方的範例中，是 UI Frontend。先前 - `package.json`：


   ```
   "devDependencies": {
       "@babel/core": "7.0.0",
       "@babel/plugin-proposal-class-properties": "7.3.3",
       "@babel/plugin-proposal-object-rest-spread": "7.3.2",
       "@material/tab-bar": "12.0.0",
       },
   ```


   之後 - `package.json`：


   ```
   "devDependencies": {
       "@types/node": "16.14.0",                     <b></b>*THIS LINE WAS ADDED
       "@babel/core": "7.0.0",
       "@babel/plugin-proposal-class-properties": "7.3.3",
       "@babel/plugin-proposal-object-rest-spread": "7.3.2",
       "@material/tab-bar": "12.0.0",
       },
   ```
