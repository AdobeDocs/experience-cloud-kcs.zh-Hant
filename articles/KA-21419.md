## 解決方法 {#resolution}

<b>請遵循以下規定的步驟：</b>
1. 位於檔案Git存放庫的根目錄中 `pom.xml,` 並尋找節點版本，例如 `nodeVersionv10.13.0/nodeVersion`


2. 找出 `ui.frontend/package.json,` 並加入已存在 `devDependencies` 物件：

   例如「devDependencies」：


   ```
   {
           "@types/node": "10.13.0",
           ...}
   ```



<br>注意：<br>


1. 不得複製 nodeVersion <b>v10.13.0</b> 中的 v。


2. 您的節點版本可能不同。例如： <b>12.22.7</b>
