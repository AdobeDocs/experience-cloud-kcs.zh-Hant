# 建置步驟期間管道部署失敗

## 說明 {#description}


<b>說明</b> :由於ui.frontend程式碼中的錯誤，在建置步驟期間管道執行失敗

在build.log檔案中，您會看到如下的ERROR訊息：

## 解決方法 {#resolution}


- 在git repo的根目錄中找出檔案pom.xml，並尋找節點版本，例如：nodeVersionv10.13.0/nodeVersion

- 找到ui.frontend/package.json ，並將新增至已存在的devDependencies物件：例如：


```
"devDependencies": {"@types/node": "10.13.0",...}
```


注意：
1. 不得複製來自nodeVersionv10.13.0的v。
2. 節點版本在您的案例中可能不同。 例如：12.22.7
